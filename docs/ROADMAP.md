# Roadmap

## Now

- Basic shell of an application with GOV.UK/DfE Frontend integration

## Next

- Implement security headers middleware
- Create mock authentication for local development
- Develop basic page CRUD functionality for admin users
- Basic storage implementation (in-memory/file-based repository)

## Later

- Tag helpers for embedding CMS content in applications
- Content versioning and audit trails
- Real authentication integration (Entra ID, DfE Sign-in)
- Database integration (Entity Framework Core with Azure SQL)
- Deployment to Azure Container Apps with Terraform
- CI/CD pipeline setup with GitHub Actions
- User roles and permissions management
- Custom content types (e.g. "Question", "Answer", "Landing Page", etc.)
- Nested content types
- Relationships
